# 監視資料庫

您的所有 Campaign 執行個體資產、工作流程和資料都儲存在資料庫中。達到最大容量的資料庫可能會導致實例出現問題（無法登錄、發送電子郵件等）。 因此，必須定期監視執行個體的資料庫，以確保最佳效能。

瞭解如何使用 [!DNL Control Panel]。
