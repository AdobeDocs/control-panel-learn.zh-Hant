# GPG 金鑰管理──概觀

GPG (GnuPG) 使用非對稱的單一金鑰組加密訊息。 若要使用 [!DNL Campaign] 實施 GPG 加密，管理員使用者必須直接從 [控制面板] 在行銷執行個體上安裝，然後/或是生成 GPG 金鑰。
瞭解如何在指定的 [!DNL Campaign] 執行個體產生和安裝公開/私用金鑰組，以加密傳出資料。 了解如何針對 [!DNL Campaign] 執行個體匯入、安裝公開金鑰，以便解密傳入資料。

* [使用 GPG 金鑰加密資料](./use-a-gpg-key-to-encrypt-data.md)
* [解密資料](./decrypt-data.md)
